# yak_runner

A Dart package that makes running functions safer and easier to test

---

### Get Started

add to your `pubspec.yaml`

```yaml
dependencies: 
  yak_runner: <latest version>
```

add to your code 

```dart
import 'package:yak_runner/yak_runner.dart';
```
