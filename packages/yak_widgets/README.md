# yak_widgets

A collection of `widgets` based on [riverpod] and [flutter_hooks]

---

### Get Started

add to your `pubspec.yaml`

```yaml
dependencies: 
  yak_widgets: <latest version>
```

---

[riverpod]: https://pub.dev/packages/riverpod
[flutter_hooks]: https://pub.dev/packages/flutter_hooks
